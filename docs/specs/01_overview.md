# 第1章: プロジェクト概要

## 1.1 プロジェクト目的

WASM, WebGPU, WebSocket を用いた高速なリアルタイムお絵かきチャットを構築し、Discord と連携することで音声通話を統合したコラボレーション環境を提供する。

## 1.2 特徴

- WASM による高速ブラシエンジン
- WebGPU による GPU 描画
- WebSocket によるリアルタイム同期
- Discord OAuth + Bot による音声連携
- GitHub Pages で動作するフロントエンド

## 1.3 想定ユーザー

- オンラインで共同作業したいクリエイター
- お絵かき配信者
- 学習や教育用途の利用者
