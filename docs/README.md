# Oekaki Chat Pages

`docs/` 配下に置いた GitHub Pages 向けの静的 UI です。

## 使い方

- `docs/index.html` は入口ページです
- `docs/demo.html` は今作っているコラボ描画の確認用画面です
- GitHub Pages の公開先を `docs/` に設定してください

## できること

- コラボ描画の確認用画面
- ローカルで描いたストロークの保存
- 相手側ストロークのサンプル追加
- フィードの確認

## 次の拡張候補

- 複数ユーザーの接続を WebSocket で同期
- ブラシ設定の拡張
- 送受信イベントの可視化
- 共有ルームの切り替え
