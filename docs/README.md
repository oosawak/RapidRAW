# RapidRAW Studio

`docs/` 配下に置いた GitHub Pages 向けの静的 UI です。

## できること

- 画像ファイルの読み込み
- 調整スライダーによる見た目の変更
- 最近開いたファイルの保存
- プリセット保存と読み込み
- PNG 書き出し

## 使い方

`docs/index.html` をブラウザで開くか、GitHub Pages の公開先を `docs/` に設定してください。

## 次の拡張候補

- RAW デコードの追加
- レイヤー編集
- マスクツール
- 非破壊編集用 `.rrdata` 相当の保存形式
