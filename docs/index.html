What this project is building

docs

このプロジェクトは、RapidRAW をベースにお絵かきチャット / コラボ描画を作る試作です。
自分が描く入力は `WASM` で補間や平滑化を行い、相手の描画は `Canvas` で表示する、という役割分担を前提にしています。
さらに、将来的には `WebSocket` で線の送受信をつなぎ、`Discord` 連携も見据えています。
