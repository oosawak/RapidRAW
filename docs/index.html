Project Overview

docs

このプロジェクトは `docs/index.html` を入口にし、`WASM Lab`、`WASM Drawing`、`RapidRAW` を別ページとして開く構成です。
`docs/wasm-lab.html` は WASM で整形した点列を Canvas に描く採用版の画面、`docs/wasm-draw.html` はこれから作る WASM 描画の確認ページ、`docs/rapidraw.html` は RapidRAW 風の確認用画面です。
描画の中身は `WASM` で処理し、ページの説明や導線は `docs/` 側で管理します。
