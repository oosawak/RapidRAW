Project Structure

docs

このプロジェクトは `docs/index.html` を入口にし、`Demo` と `RapidRAW` を別ページとして開く構成です。
`docs/demo.html` はお絵かきチャットの確認用画面、`docs/rapidraw.html` は RapidRAW 風の確認用画面です。
描画の中身は `WASM` で処理し、ページの説明や導線は `docs/` 側で管理します。

  • Entry `index.html` から各ページへ移動する
  • Demo コラボ描画の確認用画面
  • RapidRAW 写真編集プレビュー
